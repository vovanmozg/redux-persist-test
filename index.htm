Using redux-persist migration demo

  1. Open index.htm in Google Chrome
  2. Clean local storage: press "app 2 run", press "purge", reload page
  3. Start app 1: press "app 1 run"
  4. Update app: reload page
  5. Start app 2: press "app 2 run". Migration 2 should be executed
  6. Change state of app 2: press "process" near the "app 2 run" button
  7. Update app: reload page
  8. Start app 3: press "app 3 run". Migration 3 should be executed
